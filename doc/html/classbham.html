Detailed Description

template<uint8_t dx = 64>
class bham< dx >

The bham class is used for generating a single high frequency PWM output signal from a single low frequency input signal.

Parameters:
dxThe number of distinct signal levels for the input and output. Data should be conditioned to be in this range before being used with the set() method.
